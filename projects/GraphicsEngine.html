커스텀 그래픽 엔진

  • 프로젝트 설명: 3D 커스텀 그래픽 엔진 / 솔로 프로젝트
  • 기술 스택: C++, OpenGL, GLSL, ImGui
  • Github URL: Project Link

  • Deferred Shading 구현: 그래픽 엔진에 Deferred Shading을 성공적으로 구현했습니다. 이 렌더링 기법은 여러 광원에서 조명 계산을 보다 효율적으로 처리할 수 있게 하여 모든 객체가 렌더링된 후에 조명 계산을 지연시키는 방식을 사용합니다. 이를 통해 퍼포먼서 향상을 이끌어 냈습니다

  • 하이브리드 렌더링 기능: 엔진에 Deferred Shading과 Forward Shading 기법을 통합하여 하이브리드 렌더링 접근 방식을 제공했습니다. 이 하이브리드 모델은 두 가지 방법의 유연성과 강점을 제공하여 다양한 장면과 재료에 대해 향상된 렌더링 기능을 제공합니다.

  • 그림자 맵 구현: 3D 객체에 사실적인 그림자를 추가하기 위해 그림자 맵 기능을 개발했습니다. 이 기술은 광원의 관점에서 깊이 맵을 생성한 후 이를 사용하여 장면의 3D 모델에 그림자를 정확하게 계산하고 렌더링합니다.

  • Compute Shader를 사용한 가우시안 블러: 가우시안 가중치 커널이 있는 컨볼루션 필터를 사용하여 고정 너비 블러를 구현했습니다. 이 방법은 이미지를 효율적으로 블러 처리하여 부드러운 외관을 제공하며, 사실적인 부드러운 그림자와 깊이 효과를 만드는데 특히 효과적입니다.

  • 블러 필터를 사용한 모멘트 그림자 맵: 블러 필터를 사용한 모멘트 그림자 맵을 구현하여 그림자 맵핑 기술을 확장했습니다. 이 접근 방식은 그림자의 품질을 향상시키고, 아티팩트를 줄이며, 특히 복잡한 조명이 있는 장면에서 더 부드럽고 사실적인 그림자를 제공합니다.

  • 물리 기반 조명(PBL) 및 이미지 기반 조명(IBL): 물리 기반 조명(PBL)을 구현하여 사실적인 재료 상호 작용을 제공하고 이미지 기반 조명(IBL)을 구현하여 향상된 환경 반사를 제공하여 엔진의 시각적 충실도를 크게 향상시켰습니다.

  • Assimp를 사용한 3D 모델 로딩: 다양한 3D 모델 형식을 효율적으로 로드하고 처리할 수 있도록 Assimp 라이브러리를 통합했습니다.
